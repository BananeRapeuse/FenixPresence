# FenixPresence

A Python application that brings Discord and FenixPool together by displaying your mining statistics directly in your Discord status.

It shows **"Playing FenixPool"** on your profile, and the description of the Discord activity displays information about your mining activity, such as your hashrate, currently mined coin, blocks found, rewards, and more.

# Affiliation

FenixPresence is **not officially affiliated with FenixPool**.

This project is made for the FenixPool community, by a member of the community.

# Why Only FenixPool?

For now, FenixPresence is only designed to work with FenixPool.

This is because FenixPool is the only pool I am currently mining on, and it is the pool I know the best and enjoy using.

I also do not know yet how complex it would be to create a fully multi-pool compatible application, so I want to focus on making the FenixPool version solid first.
